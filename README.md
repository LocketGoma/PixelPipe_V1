# PixelPipe_V1
자체 엔진 PixelPipe Version 1 개발용. 

### 목표

상용엔진... 급은 아니라도 에디터 기반의 개발이 가능한 엔진 개발


### 기술 스택

##### Base Graphics Engine : DirectX 12
##### Base Physics Engine : PhysX
##### Base Sound Engine : Wwise
